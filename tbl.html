Tabellenrechner

This is a demo of the table predictor for a small soccer league with only four teams. Other leagues can be created by changing the teams and matchdays within the tbl-data.js file.

How to use

You can either enter the predicted results directly or:

  • Left click on a team name to increase the number of goals by one.
  • Right click on a team name to decrease the number of goals by one.
  • Left click on an empty input field to set the number of goals to zero.

The table will be updated automatically whenever a result is changed.

Clicking on the × next to a game will remove the result of the game, as if the game was not (yet) played.

Clicking on Reset will revert the table to its initial values.
